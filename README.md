# Sales Data Analysis Dashboard

## 📊 Project Overview
This project analyzes sales data to identify key business insights, profitability trends, and the impact of discount strategies.

## 🎯 Objectives
- Analyze profit trends across regions and categories
- Identify impact of discounts on profitability
- Highlight underperforming products

## 🛠 Tools Used
- Excel (Data Cleaning & Analysis)
- Tableau (Dashboard Creation)

## 🔍 Key Insights
- High discounts resulted in significant profit losses (~$135K)
- Furniture category showed lowest profitability despite high sales
- Tables sub-category generated negative profit (~$17K)
- Western region performed best, Central region underperformed

## 💡 Recommendations
- Reduce high discount ranges (above 30%)
- Focus investment on Technology category
- Optimize pricing strategy for Furniture and Tables
